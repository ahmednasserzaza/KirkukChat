# KirkukChat

Simple chat screen design built with Kotlin Jetpack Compose.

## Technology used

- Kotlin
- Jetpack Compose
- MVVM Android Architecture Pattern

## Features will be added soon ISA

- Websocket.IO
- Firebase
- Room database
